Геттеры и сеттеры
Геттеры - это такое заковыристый метод который возвращает значение приватного поля, они жестко обозначаются getX(), где Х это имя поля с заглавной буквы (позволяет прочитать значение поля вне класса)
Пример:
private String name;

public String getName() {
    return name;
}

Сеттеры - это тоже методы, но они нужны для изменения значения приватного поля, обозначается setX()
Пример:
private String name;

public void setName(String name) {
    this.name = name;
}

По факту они оба нужны для работы с инкапсулцией
